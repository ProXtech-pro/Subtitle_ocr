Place Tesseract traineddata files here (project-local tessdata_best).

Required (based on your selected TESS_LANG):
  - eng.traineddata  (English)
  - ron.traineddata  (Romanian)

Source (official):
  https://github.com/tesseract-ocr/tessdata_best

Notes:
- This project expects TESSDATA_PREFIX to point to this folder by default.
- You still need Tesseract installed (tesseract.exe).
